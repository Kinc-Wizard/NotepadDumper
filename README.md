# NotepadDumper
NotepadDumper is a small utility that reads Notepad’s temporary files from
'\AppData\Local\Packages\Microsoft.WindowsNotepad_8wekyb3d8bbwe\LocalState\TabState'.
By default, the new Notepad app stores unsaved tabs in this folder, letting you recover text that wasn’t saved.
